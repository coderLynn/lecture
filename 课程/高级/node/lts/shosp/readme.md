一.使用方法：
===================
  1.先下载shop文件夹<br>
  2.在windows+r进入命令行输入npm install --save-dev<br>
  3.然后输入npm start启动项目<br>
  4.在浏览器输入你的localhost:3000就会看到官网首页。<br>
  5.本网站兼容手机端;<br>
  6.如果遇到任何问题请加群：424734136我们会解答您的问题<br>
